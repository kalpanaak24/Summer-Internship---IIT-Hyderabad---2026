# Logic Function Realization

## Overview
This project presents the realization and analysis of a combinational logic function based on **GATE EE 2019 (Question 19)**. The Boolean expression was derived, verified using a truth table, and implemented through both Embedded C and 8051 Assembly programming.

## Features
- Boolean expression derivation
- Truth table verification
- Embedded C implementation
- 8051 Assembly implementation
- Technical report in LaTeX
